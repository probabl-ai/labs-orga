# labs-orga

This repository is used to manage project boards that help us keep track of our work at Probabl Labs.

## Purpose

The project boards in this repository serve as central hubs for tracking tasks, progress, and deliverables across different scopes:

- **Per-person boards**: Individual team members can track their personal tasks, goals, and ongoing work
- **Per-project boards**: Collaborative project boards that track tasks and milestones for specific projects or initiatives

## Using Project Boards

You can access the project boards through the "Projects" tab in this repository. Each board is configured to help visualize workflow, prioritize tasks, and maintain transparency across the team.
